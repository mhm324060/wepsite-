# 🇸🇦 القسم العربي

## اسم المشروع
**Pet Paradise**

---

## فكرة الموقع
**وصف المشروع:**  
موقع تعريفي وحجوزات لفندق فاخر مخصص للحيوانات الأليفة، مع تركيز بصري وسردي على الضيافة الراقية، العناية اليومية، الإقامة الليلية، والمرافق المتخصصة.

---

## هدف الموقع
**الهدف:**  
تقديم تجربة رقمية واضحة وراقية تساعد أصحاب الحيوانات الأليفة على التعرف على الخدمات، واستكشاف المرافق، ومقارنة الباقات، ومشاهدة المعرض، وطلب الحجز أو التواصل مع الفريق بسهولة.

---

## الصفحات الموجودة في الموقع

| الملف | الوصف |
|------|------|
| `index.html` | الصفحة الرئيسية |
| `about.html` | من نحن |
| `facilities.html` | المرافق |
| `packages.html` | الباقات |
| `gallery.html` | المعرض |
| `faq.html` | الأسئلة الشائعة |
| `book.html` | احجز الآن |
| `contact.html` | اتصل بنا |

---

## التقنيات المستخدمة

| التقنية | الاستخدام |
|--------|-----------|
| `HTML5` | بناء الصفحات والمحتوى |
| `CSS3` | تنسيق الواجهة وتقديم تجربة عرض فاخرة |
| `JavaScript` | إدارة التفاعلات والمنطق المشترك |
| `Tailwind CSS` | استخدام utility classes عبر CDN في الواجهة الحالية |
| `IntersectionObserver + CSS` | حركات خفيفة محسنة للأداء |

---

## وصف مختصر لكل صفحة

| الصفحة | الوصف |
|------|------|
| الصفحة الرئيسية | تعرض الهوية العامة للموقع، والمزايا الأساسية، والمرافق المختارة، والباقات، وآراء العملاء |
| من نحن | توضح رسالة المشروع، وفلسفة الرعاية، والخلفية التشغيلية للفريق |
| المرافق | تستعرض المساحات والخدمات التي تدعم اللعب، والراحة، والعلاج، والتنقل الآمن |
| الباقات | تعرض مستويات الإقامة الثلاث مع تفاصيل المزايا والأسعار |
| المعرض | يقدم معرضًا بصريًا منسقًا مع معاينة احترافية للصور |
| الأسئلة الشائعة | يوفر إجابات مباشرة على أكثر الاستفسارات المتعلقة بالإقامة والرعاية |
| احجز الآن | يضم نموذج الحجز مع اختيار الباقة، والتواريخ، وملاحظات الرعاية |
| اتصل بنا | يتيح وسائل التواصل، ونموذج الرسائل، وموقع الوصول عبر الخريطة |

---

## وصف تجربة المستخدم

**التجربة:**  
واجهة فاخرة وسلسة تدعم اللغة العربية واتجاه RTL بشكل كامل، مع محتوى واضح، وانتقالات هادئة، وعناصر تفاعلية منظمة، ومسار مباشر من الاستكشاف إلى الحجز أو التواصل.

---

## مميزات الموقع

- تصميم بصري فاخر ومتناسق عبر جميع الصفحات
- هيكل مشترك للترويسة والفوتر والتنقل
- دعم العربية والإنجليزية مع تبديل الاتجاه
- نماذج تفاعلية للحجز والتواصل
- عرض واضح للباقات مع ملخص حجز مباشر
- معرض صور مع نافذة معاينة Lightbox
- شريط إجراءات سريع للتواصل والحجز
- واجهة محسنة للموبايل وسطح المكتب

---

## الأداء والتجاوب مع الهاتف

**الأداء:**  
الموقع مبني كواجهة ثابتة خفيفة نسبيًا، مع تحسينات للصور، وlazy loading للوسائط غير الحرجة، ودعم للحركات التدريجية، وتخطيط متجاوب يعمل على الهاتف، والتابلت، والشاشات الكبيرة مع الحفاظ على ثبات الهوية البصرية.

---

## طريقة تشغيل الموقع

1. افتح مجلد المشروع محليًا.
2. شغّل ملف `index.html` مباشرة أو استخدم خادمًا محليًا مثل `Live Server`.
3. يُفضّل استخدام خادم محلي للحصول على تنقل أنظف بين الصفحات وتحميل صحيح للملفات.

# 🇺🇸 English Section

## Project Name
**Pet Paradise**

---

## Website Concept

**Overview:**  
Pet Paradise is a premium pet hospitality website for a luxury pet hotel in Aqaba, combining elegant presentation with booking, package comparison, facility discovery, and direct contact flows.

---

## Website Goal

**Goal:**  
The website is designed to help pet owners explore the brand, understand the care experience, compare packages, view facilities and gallery content, and submit booking or contact requests with confidence.

---

## Website Pages

| File | Description |
|-----|-------------|
| `index.html` | Home page |
| `about.html` | About page |
| `facilities.html` | Facilities page |
| `packages.html` | Packages page |
| `gallery.html` | Gallery page |
| `faq.html` | FAQ page |
| `book.html` | Booking page |
| `contact.html` | Contact page |

---

## Technologies Used

| Technology | Purpose |
|-----------|---------|
| `HTML5` | Page structure and content |
| `CSS3` | Layout, styling, and premium presentation |
| `JavaScript` | Interactivity and shared UI logic |
| `Tailwind CSS` | Utility-based styling via CDN |
| `IntersectionObserver + CSS` | Lightweight performance-friendly motion |

---

## Short Explanation of Each Page

| Page | Description |
|-----|-------------|
| Home | Introduces the brand, value proposition, featured facilities, packages, and testimonials |
| About | Explains the mission, care philosophy, and operational mindset behind the brand |
| Facilities | Showcases the spaces designed for rest, play, wellness, and movement |
| Packages | Presents the three package tiers and their included benefits |
| Gallery | Provides a curated image gallery with a professional lightbox experience |
| FAQ | Answers common questions about care, routines, and booking expectations |
| Book Now | Contains the booking request form, package selection, dates, and care notes |
| Contact | Provides direct contact methods, a message form, and map-based location access |

---

## User Experience Concept

**Experience:**  
The website is built around a polished luxury aesthetic, bilingual support, smooth navigation, clear calls to action, and a structured path from browsing to booking, while preserving strong Arabic RTL usability.

---

## Main Features

- Premium multi-page presentation
- Shared header, footer, and navigation shell
- Arabic and English language support
- RTL and LTR layout handling
- Interactive booking and contact forms
- Package comparison and live booking summary
- Gallery lightbox interaction
- Mobile-friendly quick action bar
- Smooth transitions and refined UI behavior

---

## Performance and Responsiveness

**Performance:**  
The project is delivered as a lightweight static website with responsive layouts for mobile, tablet, and desktop, image loading optimizations, lazy loading for non-critical media, and streamlined interaction behavior for smoother performance.

---

## How to Run the Project

1. Open the project folder locally.
2. Launch `index.html` directly or run the site with a local server such as `Live Server`.
3. A local server is recommended for cleaner multi-page navigation and more reliable asset loading.
