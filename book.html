# Image Asset Structure This folder is organized by asset purpose to keep handoff and maintenance simple. ## Folders - `branding/`: Brand identity assets such as the main logo. - `icons/`: Reusable SVG icons for packages and UI elements. - `showcase/`: Core property and experience photography used across primary pages. - `packages/`: Visuals dedicated to package presentation cards. - `pages/`: Page-specific hero images. - `gallery/`: Curated gallery images used in the lightbox and gallery layout. ## Naming - All filenames use lowercase kebab-case. - Names describe the asset content or role. - Shared assets are separated from page-specific assets for easier vendor handoff.
